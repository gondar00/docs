# 3 Minutes Interactive DEMO: Automating a Lead Nurture Sequence

This demonstration highlights how quickly you can set up a powerful, multi-channel lead nurture sequence using Stretch AI.

## Scenario: Converting a New Lead from a Web Form

**Goal**: Automatically send a welcome email, a follow-up SMS, and add the lead to a "New Leads" list upon form submission.

### Step 1: Create the Form (30 seconds)

1.  Navigate to **Form Builder** and click **Add Form**.
2.  Name the form "Website Lead Capture."
3.  Design the form to collect Name, Email, and Phone Number.

### Step 2: Create the Templates (60 seconds)

1.  Navigate to **Templates**.
2.  Create an **Email Template** named "Welcome to [Your Business]" using the drag-and-drop editor.
3.  Create an **SMS Template** named "Follow-up Offer" with a link to a special trial.

### Step 3: Build the Automation (90 seconds)

1.  Navigate to **Automations**.
2.  Select **Discover Automations** and choose the "Recommended automations for form submissions" template.
3.  **Trigger**: Set the trigger to "Form Submission" and select the "Website Lead Capture" form.
4.  **Action 1 (List)**: Add an action to automatically add the contact to a "New Leads" list.
5.  **Action 2 (Email)**: Add an action to send the "Welcome to [Your Business]" email template immediately.
6.  **Action 3 (Delay)**: Add a 24-hour delay.
7.  **Action 4 (SMS)**: Add an action to send the "Follow-up Offer" SMS template.
8.  **Activate**: Save and activate the automation.

## Result

In under three minutes, you have created a fully automated, multi-channel lead nurture sequence that ensures every new lead is immediately engaged and guided toward conversion.
